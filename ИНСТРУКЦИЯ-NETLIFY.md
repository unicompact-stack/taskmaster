# Инструкция: Deploy дашборда на Netlify

## Шаг 1: Регистрация через Google/GitHub

1. Открой браузер
2. Перейди на **https://app.netlify.com/signup**
3. Нажми **"GitHub"** или **"Google"** (кнопки вверху)
4. Подтверди авторизацию (введи пароль от Google/GitHub если попросит)
5. Готово — ты в аккаунте

## Шаг 2: Загрузка дашборда

1. Нажми **"Add new site"** (синяя кнопка в правом верхнем углу)
2. Выбери **"Deploy manually"**
3. Открой проводник, перейди в папку:
   ```
   C:\Users\User\Pictures\ВС код\dashboard\deploy
   ```
4. Перетащи **всю папку `deploy`** в область загрузки на сайте
5. Назови сайт: введи `taskmaster` в поле "Site name"
6. Нажми **"Save"**

## Шаг 3: Получение ссылки

1. После загрузки откроется страница сайта
2. Скопируй ссылку из адресной строки (вид: `https://taskmaster-xxxxx.netlify.app`)
3. Сохрани эту ссылку — это адрес твоего дашборда

## Шаг 4: Открытие с телефона

1. На телефоне открой браузер
2. Вставь ссылку в адресную строку
3. Дашборд загружен и работает

## Готово!

Дашборд доступен с любого устройства по этой ссылке.
